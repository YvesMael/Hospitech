# Hospitech
Suivi de la maintenance des appareils à l'hôpital

Architecture de développement : MVC

standards des commits : 

    (key) : description

les valeurs key : 
- #### feature dans le cas de l'ajout d'une nouvelle fonctionnalité
- #### fix dans le cas de la réparation d'une fonctionnalité ou d'un code cassé
- #### add_o dans le cas de l'ajout d'autres choses qu'une fonctionnalité
- #### update_o dans le cas de la modif d'autres choses qu'une fonctionnalité.

La description associée doit être compréhensible.
#### exple : (feature) : création de l'api d'ajout de nouveaux appareils
####         (fix) : changement des sessions en envoi des tokens pour faciliter la connexion
